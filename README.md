  Object Oriented Programming (OOP) adalah paradigma pemrograman yang berfokus pada penggunaan objek dan class untuk membangun sebuah program. Dalam OOP, data dan fungsi yang berkaitan digabungkan ke dalam satu kesatuan yang disebut class, sedangkan objek merupakan bentuk nyata dari class tersebut. Dengan menggunakan konsep OOP, program dapat dibuat lebih terstruktur, mudah dipahami, dan lebih mudah dikembangkan karena setiap bagian program memiliki tanggung jawab yang jelas.
  Pada program perhitungan Indeks Prestasi Semester (IPS), konsep OOP diterapkan dengan membuat class MataKuliah yang berfungsi untuk menyimpan data seperti nama mata kuliah, jumlah SKS, nilai huruf, dan bobot nilai. Class ini juga memiliki method konversiNilai() yang digunakan untuk mengubah nilai huruf menjadi bobot angka menggunakan struktur switch-case. Kemudian di dalam class Main dibuat objek dari class MataKuliah untuk memproses data yang diinput oleh pengguna. Dengan penerapan konsep OOP ini, program menjadi lebih rapi, terorganisir, dan mempermudah proses perhitungan IPS berdasarkan data mata kuliah yang dimasukkan.
